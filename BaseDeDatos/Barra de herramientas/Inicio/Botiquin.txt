Características de un botiquín genérico

El estuche debe ser :

*Resistente
*Impedir el paso del agua hacia su contenido
*Ser visible para todos
*Estar sin llave

Elementos de un botiquín :

*Jabón
*Algodón
*Parche curita
*Guantes de procedimiento
*Suero fisiológico
*Gasas
*Vendas.
*Linterna
*Termómetro
*Numeros de emergencia
*Revisar fecha de vencimiento.
*Vendas adhesivas básicas -> heridas menores
*Bajalenguas o depresor lingual -> Para deprimir la lengua y permitir el examen de la boca y la garganta
*Compresas -> Para cubrir heridas mayores
*Algodón
*Cinta micropore -> para fijar gasas y vendas
*Vendas elásticas -> Cubrir heridas mayores o dar firmeza a heridas internas
*1 caja de tabletas de ácido acetilsalicílico 500 mg. -> para dolor en general
*1 caja de tabletas Paracetamol 500 mg. -> Para el dolor en general, si el anterior provoca alergias
*Rollo de gasa auto adherente -> Para heridas
*Tijeras para cortar gasa
*Una compresa frio-caliente reutilizable (38 a 40 cm.) -> Para heridas o quemaduras 
*1 caja de antihistamínico -> para las reacciones alérgicas
